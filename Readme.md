# 🗺️ Smart Nearby Location Finder

## 📌 Overview
Smart Nearby Location Finder is a web application that allows users to discover nearby essential services such as restaurants, hospitals, and ATMs based on their current location.

The application integrates Google Maps API to provide real-time, accurate, and interactive location-based results.

---

## 🚀 Features
- 📍 Detect user’s current location
- 🍽️ Find nearby restaurants
- 🏥 Locate hospitals
- 🏧 Discover nearby ATMs
- 🗺️ Interactive map view
- ⚡ Fast and responsive UI

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask) / Node.js (optional)
- API: Google Maps API

---

## 🔑 API Used
- Google Maps JavaScript API
- Places API

## ⚙️ Installation & Setup

1. Clone the repository

git clone https://github.com/your-username/nearby-location-finder.git

cd nearby-location-finder


2. Create virtual environment

python -m venv venv
venv\Scripts\activate # Windows


3. Install dependencies

pip install -r requirements.txt


4. Add your API Key

Replace YOUR_API_KEY in your code with Google Maps API key


5. Run the application

python app.py


6. Open in browser

http://127.0.0.1:5000
